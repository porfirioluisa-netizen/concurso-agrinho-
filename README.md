# Agro Forte, Futuro Sustentável — Concurso Agrinho 2026

## 🌾 Sobre o Projeto
Este site interativo foi construído especificamente para atender à Subcategoria 3 do Concurso Agrinho de Programação 2026. O objetivo principal da plataforma é educar e conscientizar os usuários a respeito do equilíbrio fundamental entre a alta produtividade rural e a preservação ativa do meio ambiente. Através de um simulador hidropônico dinâmico, demonstramos matematicamente a preservação do recurso hídrico campo afora.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de formulários, seções organizadas e tags de acessibilidade.
* **CSS3:** Estilização com layout flexível (Flexbox), animações nativas de transição e responsividade total via Media Queries de largura.
* **JavaScript Puro (ES6):** Controle lógico de variáveis, escuta de eventos, alternância de temas e manipulação direta do Modelo de Objeto do Documento (DOM).

## 🚀 Como Executar o Projeto
1. Clone o repositório ou baixe os três arquivos principais (`index.html`, `style.css` e `script.js`) na mesma pasta.
2. Abra o arquivo `index.html` diretamente em qualquer navegador de internet atual (Chrome, Firefox, Edge, Safari).
3. Para testar o Simulador, role a página até a respectiva seção, insira seu nome, declare o peso e clique em "Calcular Equilíbrio".
4. Experimente clicar no botão de "Modo Escuro" localizado no topo para avaliar a usabilidade e o contraste visual adaptativo.
